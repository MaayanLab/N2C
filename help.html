Using the Network2Canvas Website - Download | Video

This guide provides assistance on using the N2C Website. Topics include changing the canvas display, modifying the color properties of the canvas, finding nodes by their name, running gene and drug list enrichment analyses, and downloading the displayed canvas.


Creating Your Own Canvases - Download

This guide teaches how to create a canvas of your own using the Network2Canvas script. It also provides assistance in using the other scripts included in the "Network2Canvas and Related Scripts" package.


Creating Canvases from Network Lists - Video

This video guide teaches how to create a canvas from a network list. This tutorial uses the List2Networks script found in the Network2Canvas package.


Network2Canvas and Related Scripts - Download

This zipped Python package contains the Network2Canvas script and other supporting scripts. The package also includes example input files.


Supporting Canvases and GMT Files - Download

The online canvases used on the N2C website and their associated GMT files can be downloaded from this link.
